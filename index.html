Dr. W's Zoo of Functions

Functions, like animals, come in many shapes and sizes with different exotic features. Here are some of the most basic functions with the different kinds of exotic features we will study in our class.

\(f(x) = 1\)
\(\textbf{Domain: } (-\infty, \infty)\)
The constant function, \(1\).
\(f(x) = x\)
\(\textbf{Domain: } (-\infty, \infty)\)
The linear function, \(x\).
\(f(x) = x^2\)
\(\textbf{Domain: } (-\infty, \infty)\)
The parabola, \(x^2\).
\(f(x) = x^3\)
\(\textbf{Domain: } (-\infty, \infty)\)
The cubic function, \(x^3\).
\(f(x) = \sqrt{x}\)
\(\textbf{Domain: } [0, \infty)\)
The square root function.
\(f(x) = \sqrt[3]{x}\)
\(\textbf{Domain: } (-\infty, \infty)\)
The cube root function.
\(f(x) = \sqrt[3]{x^2}\)
\(\textbf{Domain: } (-\infty, \infty)\)
The cube root squared.
\(f(x) = \frac{x}{x}\)
\(\textbf{Domain: } (-\infty, 0) \cup (0, \infty)\)
The constant function, \(1\), with a hole when \(x=0\).
\(f(x) = \frac{x^2}{x}\)
\(\textbf{Domain: } (-\infty, 0) \cup (0, \infty)\)
The linear function, \(x\), with a hole at \(x=0\).
\(f(x) = \frac{x^3}{x}\)
\(\textbf{Domain: } (-\infty, 0) \cup (0, \infty)\)
The parabola, \(x^2\), with a hole at \(x=0\).
\(f(x) = \frac{1}{x}\)
\(\textbf{Domain: } (-\infty, 0) \cup (0, \infty)\)
The reciprocal function.
\(f(x) = \frac{1}{x^2}\)
\(\textbf{Domain: } (-\infty, 0) \cup (0, \infty)\)
The reciprocal squared function.
\(f(x) = |x|\)
\(\textbf{Domain: } (-\infty, \infty)\)
The absolute value function.
\(f(x) = \frac{|x|}{x}\)
\(\textbf{Domain: } (-\infty, 0) \cup (0, \infty)\)
The sign function.
\(f(x) = \sqrt{x^2}\)
\(\textbf{Domain: } (-\infty, \infty)\)
The algebraic form of the absolute value function.
\(f(x) = (\sqrt{x})^2\)
\(\textbf{Domain: } [0, \infty)\)
The linear function, \(x\), with domain restricted to \([0,\infty)\).
\(f(x) = \sqrt{1 - x^2}\)
\(\textbf{Domain: } [-1, 1]\)
The semicircle.
\(f(x) = \sin(x)\)
\(\textbf{Domain: } (-\infty, \infty)\)
The sine function.
\(f(x) = \sqrt{x} \cdot \sqrt{-x}\)
\(\textbf{Domain: } \{0\}\)
The point.
\(f(x) = \frac{\sqrt{x}}{\sqrt{-x}}\)
\(\textbf{Domain: } \emptyset\)
The empty function.
\(f(x)=\sum_{n=0}^{\infty}\frac{1-\cos(2^{n} x)}{2^{n}}\)
\(\textbf{Domain: } (-\infty, \infty)\)
The Blancmange Function.
\(f(x) = \sin\left(\frac{1}{x}\right)\)
\(\textbf{Domain: } (-\infty, 0) \cup (0, \infty)\)
The Topologists Sine Curve.
\(f(x) = x\sin\left(\frac{1}{x}\right)\)
\(\textbf{Domain: } (-\infty, 0) \cup (0, \infty)\)
The Shrinking Topologists Sine Curve.
\(f(x)=\sqrt{|x|}+\sin(\pi^{3}x)\sqrt{\frac{e^{2}-x^{2}}{2}}\)
\(\textbf{Domain: } [-e, e]\)
The Heartbeat Function.
